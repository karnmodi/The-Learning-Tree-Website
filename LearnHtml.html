HTML
HTML, short for HyperText Markup Language, is the foundation of all web pages. It was created by Tim Berners-Lee in 1993 to define the structure of a web page. Without it, you wouldn’t be able to organize text or add images or videos to your pages. HTML is the beginning of everything you need to know to make your first website
HTML Structure
HTML is organized into a family tree structure. HTML elements can have parents, grandparents, siblings, children, grandchildren, etc.
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>

    <body>
        <h1>My First Heading</h1>
        <p>My First Paragraph</p>
    </body>
</html>
The <!DOCTYPE> declaration tells the browser that a given file is a markup .html file and that a set of web standards must be applied to it. It is the first line of code that goes at the top of every HTML or XHTML file.
Note:   <!DOCTYPE> is not an element or a tag.
Basics of HTML
HTML Basics Includes details of div, Class, Id, and some tags which are useful in every HTML Code.
  1. HTML uses tags to define elements within a webpage.
  2. Elements are the building blocks of HTML documents, made up of tags and their content.
  3. Attributes provide additional information about an element and are used within the opening tag of an element.
  4. HTML follows a hierarchical structure, with a top-level <html> element containing the <head> and <body> elements.
  5. A DTD declares the version of HTML being used, and comments can be used to annotate code.
Classes & Ids
Classes are HTML attributes used to select one or more elements for CSS styling or JavaScript purposes.
For Styling

The class attribute is commonly used for assigning styles to one or more elements. Multiple classes can be assigned to a single element in a space-separated list. (e.g. class="class1 class2 class3").

Inside the stylesheet, a class name is selected with the following syntax:
.class-name {
}

#id-name {
}
A class selector is a name preceded by a period (.).

Below is some markup with an outer wrapper-div, containing a level 1 heading and three level 2 sub-headings. The styling for the heading will be unique. However, all elements with the sub-heading class name received the same styling. 
<!DOCTYPE html>
    <html lang="en">
        <head>
            <style>
                .heading {
                    background-color: lightcoral;
                }
                #sub-heading {
                    background-color: lightskyblue;
                    margin: 5px;
                }      
            </style>

            <title>Classes</title>
        </head>

        <body>
            <div class="wrapper-div">
                <h1 class="heading"> Welcome to Class! </h1>
        
                <h2 id="sub-heading">_</h2>
            </div>
        </body>
    </html>
This Provided code contains classes & ids in <div> <h1> and <h2> tag. Which gives an idea about How can we use these tags, Classes and Ids.

HTML Tables
HTML tables allow web developers to arrange data into rows and columns with cell's individual effects and Cellspacing.


Example of HTML Table 
<!DOCTYPE html>
    <html lang="en">
        <head>
            <title>HTML Tables</title>
        </head>

        <body>
            <table border="5px" style="text-align: center;">
                <tr>
                    <th>Company</th>
                    <th>Contract</th>
                    <th>Country</th>
                </tr>
                <tr>
                    <td>Alfreds Futterkiste</td>
                    <td>Maria Anders</td>
                    <td>Germany</td>
                </tr>
                <tr>                    
                    <td>Centro comercial Moctezuma</td>
                    <td>Francisco Chang</td>
                    <td>Mexico</td>
                </tr>
          </table>
        </body>
    </html>


This Provided code contains a table which shows the Header row of Company, Contract and Country which is in <th> tag. and rest of the cells are in <td> tag. All the rows of the table are having <tr> tag which represent the rows of the Table.

HTML Forms
HTML Forms allow web developers to create forms to get data from the User such as Username, Password, Text, Email, & many more.


Inside the stylesheet, a class name is selected with the following syntax: 
<style>
    .search-container {
        float: right;
    }
                                    
    button[type=submit] {
        background-color: #4CAF50;
        color: white;
        padding: 8px 20px;
        border: none;
        border-radius: 5px;
        cursor: pointer;
    }
                                    
    button[type=clear] {
        background-color: #ff0000;
        color: white;
        padding: 8px 20px;
        border: none;
        border-radius: 5px;
        cursor: pointer;
    }
</style>

The search bar is placed in the top right corner of the page using CSS, and the form includes a "Submit" button to send the form data to a server, and a "Clear" button to reset the form. The form uses the "post" method to send data to the server and the "required" attribute to ensure that the username, password, and email fields are filled out before submitting the form.
